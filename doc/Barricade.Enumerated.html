Mixin: Enumerated

Barricade. Enumerated

Defines a constraint on the possible values a Barricade object can take. Enums can be defined simply as an array of values or an array of objects of the form {label: someLabel, value: someValue}.

getEnumLabels() → {Array}

Returns an array of labels. If the enum has defined labels, those are returned. If the enum is simply a set of values, the values are returned as the labels.
